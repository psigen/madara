Detailed Description

template<typename T>
class Madara::Utility::LQueue_Iterator< T >

Implements a forward iterator for LQueue type classes.

Note: Having a const Expression_Tree_Iterator does not guarantee that the current *position* that it points to will not change, it only guarantees that you cannot change the underlying queue!
