void Madara::KATS::Test_Framework::barriered_sync ( const std::string &  event_name)

Syncs on all processes until everyone is at the event and has the latest information.

This function does three main operations: 1) barrier on every process synchronizing on this event, 2) marking all globals as modified to current clock, and 3) barrier on every process sending their updates.

Parameters:
event_namename of the sync event to barrier on
